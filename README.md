# TokoPlay

TokoPlay is Tokopedia Play clone for my learning purpose.

## Clone this repo
```bash
git clone https://github.com/Astrayesa/toko-play.git
cd toko-play
```


## Installation

Use the package manager [npm](https://www.npmjs.com/) to build and run TokoPlay.

```bash
npm install
```

## Run

```bash
npm start
```
